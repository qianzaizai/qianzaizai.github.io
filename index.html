介绍

Hexo 是一款快速、简单、强大的博客框架，基于 Node.js 构建。

特点：

  • 超级快速。
  • 支持 Github Markdown 和 Octopress 插件。
  • 一个命令就能部署到 GitHub Pages, Heroku 等。
  • 强大的API ，极易扩展。
  • 成百上千的主题和插件。

安装

安装 Hexo 相当简单，只需要先安装下列应用程序即可：

  • Git
