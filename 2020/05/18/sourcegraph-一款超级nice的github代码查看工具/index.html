sourcegraph__一款超级nice的github代码查看工具

Posted on 2020-05-18

身为一名程序媛，必须得跟海绵宝宝一样不断得汲取知识。

GitHub 作为全球最大的程序员聚集地，已经成为开发学习技能的绝佳伴侣。

由于是在线应用阅读代码超级不方便，然后加上网络原因一页一页的去翻看代码很煎熬的，相信在 GitHub 上面阅读过代码的童鞋肯定深有体会。

想要深入了解代码的童鞋，一般都会 git clone 到本地，再把代码导入到 IDE 才能愉快的阅读代码，但是有时候只是想快速的了解下这个项目源码，每次都克隆到本地会有点点嫌麻烦。

下面就给大家分享一个超级 nice 的插件 Sourcegraph，是开发者的福音呀！

安装 Sourcegraph 插件

  1. 在 chrome 网上应用商店中搜索：sourcegraph

百度网盘：

链接：https://pan.baidu.com/s/10KnmcNBtUJYgJFUaDM9xhg
提取码：oxru

  1. 找到 Sourcegraph for GitHub，点击添加至Chrome

在 GitHub 上利用 Sourcegraph 查看代码

进入 GitHub，找一个项目，点开一个文件，在工具栏里会看到多了一个 View File 的按钮，点击此按钮，如下图：

进入如下页面：

Find references：

您还可以查看所有调用函数的地方，可访问属性等，就像在强大的 IDE 中一样。

Go to definition:

在支持的语言中，可以转到符号的定义处，甚至可以跨仓库跳转到一个定义。
