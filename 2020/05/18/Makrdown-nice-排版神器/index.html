Makrdown_nice__排版神器

Posted on 2020-05-18

Hi ~ 大家好，这里是黔在在！

在在一直在用的一款可以排版 Markdown 语法的并且与微信公众号相结合的神器 —— Markdown Nice

工具地址 ： https://mdnice.com

简介

  • 一款开源的 Markdown 编辑器，写完即排版成功。
  • 复制即可粘贴到微信公众号、知乎和掘金。
  • 17 款主题并且支持自定义样式。
  • 8 款代码风格。
  • 内容和自定义样式浏览器中实时保存。
  • 支持上传图片、脚注、公式。
  • 颜值高。

导航栏

文件

  • 导出 Markdown
  • 导出 PDF
  • 导入电脑本地文件：可导入文本文件或者 Markdown 文件。

格式

  • 删除线
  • 加粗
  • 倾斜
  • 代码
  • 行内代码
  • 链接：可以添加在线链接
  • 表格：还可以添加表格
  • 图片：添加图片
  • 衬线字体
  • 微信外链转脚注
  • 格式化文档

功能

  • 重置：恢复所有默认设置，也会清除自定义样式，请注意！
  • 查找：查找并替换相关文本。
  • 本地历史：可以查看历史编辑情况，可以快速恢复之前的内容。
  • SitDown：用于将各个平台的富文本转换成 Markdown。

主题

17款内容主题，总有以一款你喜欢的。

可以一键切换成自己喜欢的主题样式，并且还支持自定义样式加入自己的特色。

  • 橙心
  • 墨黑
  • 姹紫
  • 嫩青
  • 绿意
  • 红绯
  • WeChat-Format
  • 蓝莹
  • 科技蓝
  • 兰青
  • 山吹
  • 前端之巅同款
  • 极客黑
  • 简
  • 蔷薇紫
  • 萌绿
  • 全栈蓝

我真的好喜欢 ==姹紫== 这个主题 ，就是少女生满满的呀！看着就很浪漫。

代码主题

7 款内置代码主题风格，流行的都有，个人觉得默认即可，还支持 Mac 风格。

  • 微信代码主题
  • atom-one-dark
  • atom-one-light
  • monokai
  • github
  • vs2015
  • xcode

设置

  • 同步滚动：顾名思义，左侧编辑区和右侧预览区同时滚动。
  • 上传图片时包含名称：保留原图片名称。

侧边栏

  • 复制到公众号
  • 复制到知乎
  • 复制到掘金
  • 手机和电脑预览模式切换。

其他

mdnice 作为一款开源产品还支持大家自己部署和二次开发。

项目地址：https://github.com/mdnice/markdown-nice
